# Stealth
Antoine Bastos and Yann Roux--Daugrois

## Install / Uninstall

```sh
make install
```

## Documentation
```sh
make doc
```
then see doc/html/index.html

## Run
```sh
make
``` 
or 
```sh
make run
``` 

## Controls

escape to exit a menu or end a game

### Move
wqsd for moving in the room

### Abilities
speed : left or right shift 
invisibility : space
